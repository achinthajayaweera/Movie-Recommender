# Movie Recommender - Full Stack ML App

A content-based movie recommendation system built with FastAPI (backend), React (frontend), and Scikit-learn (ML).

## Project Structure

```
movie-recommender/
├── backend/
│   ├── main.py
│   └── requirements.txt
├── frontend/
│   └── src/
│       ├── App.js
│       └── index.js
├── .gitignore
└── README.md
```

## Tech Stack
- **Frontend**: React.js
- **Backend**: FastAPI (Python)
- **ML**: Scikit-learn (Cosine Similarity)
- **Data**: TMDB 5000 Movies Dataset
